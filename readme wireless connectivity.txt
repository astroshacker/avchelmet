Wireless Connection Readme
--------------------------

Connect myRIO to USB:
in MAX: connect to wireless network (might need to click "Restart")
dchp or link local
save
connect to same wireless network on computer wifi
take down ip address from MAX
change my computer and myRio addresses to this address in project tree
disconnect usb, then power source
connect new power source
turn on myRio wifi (make sure led blue)
run main_myrio.vi
remove USB

To Connect IP Camera:
turn on wifi hotspot from iphone
run through same process as above
connect ip camera to iphone hotspot
choose camera from IMAQ palette

To Reconnect with USB:
Get new ip address from max ethernet
Update PC address with Rio ethernet address
